# RLN React TypeScript Waku App

**Demonstrates**:

- RLN
- React/TypeScript

The `master` branch's HEAD is deployed at https://examples.waku.org/rln-react-ts/.

To run a development version locally, do:

```shell
git clone https://github.com/waku-org/js-waku-examples
cd rln-react-ts
npm install
npm run start
```
